# **App Name**: Mindscape

## Core Features:

- User Authentication: Secure user authentication system with email/password and Google OAuth.
- Animated Dashboard: Futuristic dashboard with mood tracking, stress meter, and mindfulness tips.
- AI Chat Support: Gemini AI powered empathetic assistant for real-time chat support. The AI uses tool use for the incorporation of emergency information if self-harm or danger is mentioned.
- Emergency Helpline Display: Display of government emergency helplines for immediate access.
- Mood Tracking: Allow users to track their mood over time with emoji and a timeline.
- Chat History: Persistent chat history saved for each user.

## Style Guidelines:

- Primary color: Vibrant purple (#9D4EDD) for a futuristic and calming feel, reflecting the blend of technology and mental wellness.
- Background color: Dark desaturated purple (#1E1E24) to provide contrast and a dark mode experience.
- Accent color: Electric blue (#5BC0BE) to highlight interactive elements and create visual interest.
- Font: 'Space Grotesk' (sans-serif) for headlines and 'Inter' (sans-serif) for body text. 'Space Grotesk' gives a techy, modern feel for prominent text, while 'Inter' ensures readability for longer content.
- Code font: 'Source Code Pro' for any code snippets.
- Glassmorphism panels with soft glow edges and floating cards for a futuristic interface.
- Smooth transitions and animated hover glows for an engaging user experience.
